# Задание 1

## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для последовательности. На основе списка, содержащего цифры от 1 до 9, получить список из их двоичных значений.

### Алгоритм решения

Водим данные о количестве элементов и сами элементы последовательности. Проверяем введенные данные. С помощью рекурсии создаем последовательность. Далее к последователности с помощью .map применяем функцию перевода в двоичную СС и на основе этого генерируем новую последовательность. Далее выводим результат.

# Задание 2

## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для последовательности.  Список содержит десятичные цифры. Получить строку, перечисляющую эти цифры. Например, для списка [1; 3; 1; 7] получаем строку «один; три; один; семь».

### Алгоритм решения

Запрашиваем данные о количестве элементов и сами элементы последовательности. Проверяем введенные данные. С помощью рекурсии создаем последовательность. С помощью .fold переводим цифры в строковое представление, поэлементно "склеиваем" подстроки в итоговую строку. Выводим результат

# Задание 3

## Задача 1

### Текст задачи

Вывести список файлов, расширение которых начинается с заданного символа, в указанном каталоге.

### Алгоритм решения

Запрашиваем путь и первый символ расширения у пользователя, делаем проверку. Получаем последовательность файлов в папке. Фильтруем данную последовательность, сверяя первую букву расширения каждого файла с введенной пользователем буквой. Передаем прошедшие проверку файлы вперед, преобразуем их в удобочитаемый вид и выводим готовую последовательность.
